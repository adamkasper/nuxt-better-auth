## Quick Start

Install the module and its peer dependencies:

```bash
pnpm add @onmax/nuxt-better-auth better-auth
```

Add the module to your `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  modules: ['@onmax/nuxt-better-auth'],
})
```

## Configuration

Create your Better Auth configuration file at `server/auth.config.ts`:

```ts
import { betterAuth } from 'better-auth'

export const auth = betterAuth({
  // Your Better Auth configuration
})
```

The module will automatically detect this file and configure the auth client and server utilities.
