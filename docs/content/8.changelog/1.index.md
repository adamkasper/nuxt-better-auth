## 0.1.0

Initial release of Nuxt Better Auth.

**Features:**
- Automatic route protection via `routeRules` and `definePageMeta`
- Reactive session management with `useUserSession()`
- NuxtHub integration with schema generation
- DevTools panel for development
- Full TypeScript inference from Better Auth plugins
